# ⚡ Power System Load Flow Analysis

## 📌 Overview
This project demonstrates basic load flow analysis of a multi-bus power system to study voltage levels and system performance.

## 🔍 Features
- Bus voltage calculation
- Power flow analysis
- Multi-bus system evaluation

## 🛠️ Tools Used
- MATLAB
- Python

## 📊 Sample Output
Bus1 Voltage: 1.0 pu  
Bus2 Voltage: 0.98 pu  
Bus3 Voltage: 0.95 pu

## 🎯 Objective
To analyze how electrical power flows through a system and evaluate system stability.
